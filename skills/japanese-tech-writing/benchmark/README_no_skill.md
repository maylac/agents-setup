# interpret-paper 📄✨

**読みたい論文、もう積んでいませんか？**

interpret-paper は、論文の URL を渡すだけで、AI があなたの代わりに論文を読み解き、わかりやすい日本語の解説 HTML をサクッと生成してくれる、研究者にとって非常に強力なツールです。難解な数式や証明も、まるで専門家が隣で教えてくれるかのように、ていねいに噛み砕いて解説します。

重要なのは、面倒な設定が一切不要だという点です。Claude Code のスラッシュコマンドとしてシームレスに動作するため、あなたの研究ワークフローを劇的に加速させます。

## ✨ 主な特徴

- 🚀 **圧倒的な手軽さ**: URL を渡すだけで、あとは AI におまかせ
- 🧠 **深い理解をサポート**: 定理・証明・アルゴリズムまで徹底的に深掘り
- 🎨 **美しい出力**: 数式が美しくレンダリングされた HTML を自動生成
- 🔧 **柔軟なカスタマイズ**: 解説の詳細度を自由自在にコントロール

## 必要なもの

本ツールを利用するにあたって、以下が不可欠となります。

- Claude Code（Claudeへのアクセス権が必須です）
- uv（Pythonパッケージ管理ツール）

## セットアップ

セットアップはとても簡単です。以下のコマンドを実行するだけで、すぐに使い始めることができます。

```bash
git clone https://github.com/<your-username>/interpret-paper.git
cd interpret-paper
uv sync
```

その後、Claude Code でこのディレクトリを開けば、`/interpret` コマンドが自動的に登録されます。

## 使い方

使い方も直感的そのものです。Claude Code のチャット入力欄で `/interpret` と入力するだけで、あとはツールがよしなに処理してくれます。

```
/interpret https://arxiv.org/abs/1706.03762
/interpret papers/adam.pdf
/interpret --select
```

もちろん、自然言語でもOK。「この論文を解説して」と話しかけるだけで、AI がその意図を汲み取って動作します。

なお、arXiv の URL を渡す場合は abs ページを指定するのがおすすめです。HTML版とPDF版を賢く使い分けて、最適なコンテンツを自動で取得します。

## 出力

生成された解説は `output/` に HTML として出力されます。ブラウザで開くだけで、美しい数式がレンダリングされ、まるで論文誌のような仕上がりを体験できます（MathJax を利用するため、インターネット接続が必要です）。

## さあ、始めましょう！
